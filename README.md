Proof-of-work algorithm:
 - Symbol: GIARCOC
 - Scrypt: Nfactor, N starting at 2048)
 - Maxcoins: 50,000,000
 - Premine: 0
 - 1 minute block targets
 - Subsidy initial reward: 50
  - block 0-200 : 5 reward
 - Subsidy halves in 420,000 blocks 
 - Confirmation needed: 12 
 - COINBASE_Maturity: 100
 - KGW implementation
